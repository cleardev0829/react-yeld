# spacepup-frontend

## Getting Started

spacepup-fronted is frontend template powered by reactjs

## Available Scripts

### Running the development server.

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build
```

### Running the production server.

```bash
    npm run start
```

### Format code-base

```bash
    npm run format
```
